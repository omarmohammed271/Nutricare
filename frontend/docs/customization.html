Customize Layout

Open src/states/useLayoutContext.tsx file, and search for declaration of INIT_STATE. Here as a default theme property

Property Description
{theme:'light'} or
{theme:'dark'} 		Changes the default theme mode for the App, you can switch between dark or light
{sidenav:{mode:'mobile'}} or
{sidenav:{mode:'default'}} 		Changes the default mode of the Leftside menu, you can switch between mobile or default
{sidenav:{theme:'light'}} or
{sidenav:{theme:'dark'}} 		Changes the default theme mode left the Leftside menu, you can switch between dark or light
