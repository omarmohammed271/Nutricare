Introduction

Attex is a fully featured premium admin template built on top of awesome MUI v5.14. It's crafted using modern web technologies HTML5 and Typescript. The theme includes beautiful dashboards, many ready to use hand crafted components, widgets, pages, etc. The components can be used very easily on any page. The theme is fully responsive and customizable.

It is extremely powerful and adaptable, making it ideal for web development. applications including CRM, CMS, admin panels, dashboards, etc. It is fully responsive and has a perfect look and feel on mobiles and tablets. Also, we tried to implement this by Keeping the concept of modularity in mind, this enables you to reduce the development time and cost.

This documentation is guidelines for all users who can even be a beginner or experienced web developer.

We really care for our buyers, so in case you are having any question or feedback, please feel free to get back to us via email support@coderthemes.com or by filling out the contact form on our website.

Additional features :

  • Lazy Loading(React.lazy)
  • Custom Hooks

Structure

When you extract the zip file you received after purchasing, you will find the following files and folders: 

                                    
                                      ├── /docs
                                      ├── package.json
                                      ├── bun.lockb
                                      ├── yarn.lock
                                      ├── package-lock.json
                                      ├── vite.config.ts
                                      ├── tsconfig.json
                                      ├── tsconfig.node.json
                                      ├── index.html
                                      ├── global.d.ts
                                      ├── .prettierrc.json
                                      ├── .gitignore
                                      ├── /src
                                      │   └── /assets
                                      │       └── /css                  -- contains styles for overwriting library's default styles
                                      │       └── /images               -- contains images used throughout the app
                                      │   └── /common                   -- contains commonly used items
                                      │       └── fake-backend.ts
                                      │       └── menu-items.ts
                                      │   └── /components               -- contains commonly used components
                                      │       └── /FileUploader
                                      │       └── /form
                                      │       └── /VectorMap
                                      │       └── ...other components
                                      │   └── /helpers
                                      │   └── /hooks                    -- contains custom made hooks
                                      │   └── /layouts                  -- contains page layout
                                      │   └── /pages                    -- all pages
                                      │   └── /routes                   -- routes
                                      │   └── /states                   -- global states
                                      │   └── /themes                   -- MUI's overwritten theme
                                      │   └── /types                    -- types
                                      │   └── /utils                    -- custom utilities
                                      │   └── App.tsx
                                      │   └── main.tsx
                                      │   └── vite-env.d.ts
